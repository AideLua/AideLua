## 使用须知
  1. 本软件默认开启自动保存代码且无法关闭（自动保存触发条件：切换到其他应用、点击二次打包以及打包运行、打开其他文件、关闭文件、打开侧滑（大屏除外）、点击标签栏等）

  2. 此软件不能用来开发大型项目

  3. 此软件必须搭配AndroLua等编译器，使用第三方编译器是为了提供编译环境
