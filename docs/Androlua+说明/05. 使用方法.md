使用 Java 方法和 Java 中大体相同，更多细节可以网络搜索 `安卓API`。
## 比如
设置按钮文字
``` lua
button.setText("按钮")
```

获取按钮文字
``` lua
local text=button.getText()
```

## getter / setter
Java 的 `getxxx` / `isxxx` 方法没有参数与 `setxxx` 方法只有一个参数时可以简写
``` lua
button.text="按钮"
-- 相当于 button.setText("按钮")
local text=button.text
-- 相当于 local text=button.getText()
```
